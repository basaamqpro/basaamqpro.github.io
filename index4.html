How to update (teacher)

Manual HTML only
1) Copy a .lesson block to add a class.
2) Change data-start to the new teacher time (YYYY-MM-DDTHH:MM).
3) Change the lesson title and join link.
4) Click “Mark updated” so students see the latest update time.
Note: Timezone abbreviations like “GMT/BST” change with daylight saving. This page uses real timezone rules automatically.
